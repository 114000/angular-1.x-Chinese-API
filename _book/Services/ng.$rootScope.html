Angular_1.4.3 API 服务篇 $rootScope

  • $rootScopeProvider
  • ng 模块中的服务

每个应用都有一个单独的根作用域。所有在这个应用的其他的作用域都是它的子代作用域。作用域通过一个监听模型变化的机制来为在视图和模型之间提供分离功能。他们还可以提供事件的冒泡/捕获和和订阅功能。详情参见scope 的开发文档。
